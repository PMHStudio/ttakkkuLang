# ttakkku Lang File Format

4    8        8
xxxx yyyyyyyy zzzzzzzz

CMD  Arg딱     Arg구

* 한줄에 각각 4자, 8자, 8자. 총 20자의 "딱" 혹은 "구"가 있어야 합니다
* x, y, z 사이의 공백은 중요하지 않습니다
* CMD는 실행할 명령을 정의하고 Arg딱과 Arg구은 특정한 값을 나타낼 수 있습니다

# Commands
## 딱딱딱딱 : PNT
이 CMD값은 "가리키기"를 의미 합니다.
Arg딱값은 가리킬 메모리의 주소를 의미합니다

## 딱딱딱구 : SET
이 CMD값은 "대입"을 의미 합니다
이 줄의 Arg딱값은 대입할 숫자값을 의미하고 (값이 딱일경우 Arg구을 사용합니다)
Arg구값은 대입할 숫자의 주소값을 의미합니다

## 딱딱구딱 : ADD
이 CMD값은 "추가"를 의미 합니다
이 줄의 Arg딱값은 추가할 숫자값을 의미하고 (값이 딱일경우 Arg구을 사용합니다)
Arg구값은 추가할 숫자의 주소값을 의미합니다

## 딱딱구구 : SUB
이 CMD값은 "가감"을 의미 합니다
이 줄의 Arg딱값은 가감할 숫자값을 의미하고 (값이 딱일경우 Arg구을 사용합니다)
Arg구값은 가감할 숫자의 주소값을 의미합니다

## 딱구딱딱 : DIS
이 CMD값은 "출력"을 의미 합니다
이 줄의 Arg딱, Arg구값은 출력할 메모리의 주소를 의미하고
두 메모리 주소 사이에 있는 모든 값을 출력합니다

## 딱구딱구 : DIX
이 CMD값은 "문자출력"을 의미 합니다
이 줄의 Arg딱, Arg구값은 출력할 메모리의 주소를 의미하고
두 메모리 주소 사이에 있는 모든 값을 출력합니다

## 구구구구 : END
이 CMD값은 "중단"을 의미 합니다
이 줄의 Arg딱값은 중단시 반환할 코드를 의미하고
Arg구값은 무시됩니다
